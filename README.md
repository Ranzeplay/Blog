# Blog

A blog application for myself.

## Directory structure

```
- Root
    fallback.png
    - Articles
        - Article1
            - text.md
            - head.png
            - assets
                - ...
        - Article2
        - ...
    - Pages
        - Page1
        - Page2
        - ...
```

- `fallback.png`

  Show this image as the head image when `head.png` doesn't found in the directory.

- `head.png`

  The head image of an article.

- `text.md`

  The body text of the article.
